To see the Google Now Feed on the left, and make full use of At A Glance, an additional app is necessary. Nova Launcher uses Nova Companion, and Lawnchair uses Lawnfeed for this purpose. This is due to restrictions from Google and specifically the Google App.

All of my older GitHub builds until version 3.7 included all the functionality in one app. However, this meant a Play Store release was impossible. If I wanted to make this possible, I had to follow Nova and Lawnchair and my own additional app that would make it work. So I started programming. I called my app Rootless Pixel Bridge, similar to the launcher's name.

Installation Instructions
1. Go to https://github.com/amirzaidi/AIDLBridge/releases/latest
2. Click "pixelbridge.apk"
3. Let your phone download the file
4. Install the file as an application
5. Go into the settings of the launcher
6. Enable "Display Google app"

Some people, like Pixel and Nokia users, might not be able to install this. They will get an "App not installed" error. This happens because these people have the real Pixel Launcher installed as a system app. The Pixel Launcher conflicts with my Rootless Pixel Bridge, and cannot be installed at the same time. My older versions of Rootless Pixel Launcher can also give this error on other devices. Make sure all third party Pixel Launchers are removed from your device before trying to install the Rootless Pixel Bridge.
